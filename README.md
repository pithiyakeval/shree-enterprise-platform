# Shree Enterprise Platform

Full Stack Solar & Mandap Service Platform with AI Chatbot.

## Features

• Admin dashboard  
• Lead management  
• Solar service requests  
• Mandap service requests  
• AI chatbot support  
• CSV export  
• Analytics dashboard  

## Tech Stack

Frontend:
React
TypeScript
TailwindCSS
Vite

Backend:
FastAPI
PostgreSQL
JWT Authentication

Deployment:
Vercel (Frontend)
Render (Backend)
